### 论文贡献
#### 全卷积层取代全连接层
1. **全连接层**弊端：
   1. 输入图像尺寸维度固定
   2. 丢失了空间信息
2. **全卷积层**优势：
   1. 维度没有限制
   2. 保留了空间信息
   3. 参数与计算量同时大幅减少（无论是infer（前向传播）还是train（后向传播）时，速度快了5倍以上）
   4. 解决了全连接层没法用于pixel-wise任务的困难
